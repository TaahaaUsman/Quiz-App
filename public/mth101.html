Lecture 1: Coordinates, Graphs, and Lines

📘1. What is Calculus?

  • Calculus is a part of mathematics that studies how things change.
  • It helps us understand how one quantity changes when another quantity changes.
  • Examples: A car is moving — we use calculus to know how fast it is going at any time.

🔢2. Real Numbers

  • Natural Numbers (ℕ): 1, 2, 3, ...
  • Whole Numbers: 0, 1, 2, 3, ...
  • Integers (ℤ): ..., -3, -2, -1, 0, 1, 2, ...
  • Rational Numbers (ℚ): Can be written as a fraction (p/q). Example: 1/2, -3/4
  • Irrational Numbers: Cannot be written as a fraction. Decimal never ends or repeats. Example: √2, π

🧠3. What is a Set?

  • A set is a group or collection of things. Example: {a, e, i, o, u}
  • Subset: A set that has only some or all elements from another set. A ⊆ B

⚠️4. Why Can't We Divide by Zero?

  • Dividing by 0 is not allowed in math.
  • It gives no clear answer or wrong answers.
  • So, it's called undefined.

🔍5. Discovery of Irrational Numbers

  • Hippasus, a Greek mathematician, proved that √2 is not rational.
  • This means some numbers cannot be written as fractions.

📉6. Number Line and Coordinates

  • In the 1600s, French thinker Descartes joined math and geometry.
  • He introduced the number line.
  • Left of 0 = negative numbers, Right of 0 = positive numbers.
  • This was the start of analytic geometry, drawing graphs from equations.

📘 Lecture 2: Absolute Value

Absolute value tells us how far a number is from zero on the number line. It is always positive or zero. We write it like this: |a|.

🧠 What does it mean?

If a number is already positive or zero, we don’t change anything. But if it is negative, we remove the minus sign.

➡️ Definition:

  • If a ≥ 0, then |a| = a
  • If a < 0, then |a| = -a

➡️ Easy Examples:

  • |5| = 5
  • |-4/7| = 4/7
  • |0| = 0

🧩 Solving Equations

Like |x - 3| = 4, gives:

  • x - 3 = 4 → x = 7
  • x - 3 = -4 → x = -1

🧮 Square Root with Absolute

√(a²) = |a| – always gives a positive answer.

⚙️ Properties

  • |-a| = |a|
  • |a × b| = |a| × |b|
  • |a / b| = |a| / |b|
  • |a1 × a2 × ... × an| = |a1| × ... × |an|

📏 Geometry Use

Distance between a and b = |b - a|

Lecture 3: Coordinate Planes and Graphs

In this lecture, we will learn about:

  • Graphs in the coordinate plane
  • Intercepts
  • Symmetry

What is a Coordinate Plane?

A coordinate plane is made when two number lines cross each other at 90 degrees. These lines are called the x-axis and y-axis.

Every point on this plane can be shown using a pair of numbers, like (4, 3). These are called ordered pairs.

How to Plot a Point

To draw a point (a, b) on the plane, move "a" units on the x-axis and "b" units on the y-axis.

For example, the point (2, 5) means: move 2 steps right and 5 steps up.

What is the XY-Plane?

We usually name the horizontal line as x and the vertical line as y. So the plane becomes the "xy-plane". But we can also use other letters like u and v.

Quadrants

The coordinate plane is divided into 4 areas called "quadrants". They are numbered with Roman numbers in a counter-clockwise direction: I, II, III, IV.

Solutions of Equations

If a pair like (3, 2) satisfies an equation like 6x − 4y = 10, it is called a solution.

To check, just put x = 3 and y = 2 in the equation. If it becomes true, then it is a solution.

What is a Graph of an Equation?

The graph is all the points (x, y) that make the equation true.

Example: For the equation y = x/2, the graph will be a line showing points like (2, 1), (4, 2), etc.

Intercepts

These are the points where the graph touches the x-axis or y-axis.

  • If y = 0, the point is on x-axis → x-intercept
  • If x = 0, the point is on y-axis → y-intercept

Symmetry

Symmetry means the graph looks the same on both sides.

  • Symmetry about x-axis: (x, y) → (x, -y)
  • Symmetry about y-axis: (x, y) → (-x, y)
  • Symmetry about origin: (x, y) → (-x, -y)

Why is Symmetry Useful?

If a graph is symmetric, we only need to draw one part. The other part can be made by copying with a mirror image.

Example: Graph of y = (x² - 21x)/8

This equation is symmetric about the y-axis. So we only need to find values for x ≥ 0. The values for x < 0 will be same y-values but x will be negative.
